Pour installer le serveur en local via XAMPP:

- 1. Démarrer le logiciel XAMPP.
- 2. Démarrer un serveur Apache et MySQL.   
- 3. Autorisez l'accès aux bons ports.
- 4. Insérer tous les fichiers contenu dans le .zip dans c:\xampp\htdocs.
- 5. Dans la barre de recherche de votre navigateur, entrez l'adresse "http://localhost/index.html".

Et voilà !
